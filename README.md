# VeryTontine (Sui Move)

VeryTontine is a community savings (tontine / ikirimba) protocol built on Sui.

## Goal
Enable trusted groups and diaspora communities to contribute funds transparently
using smart contracts.

## MVP Scope (Mission 1)
- Initialize Sui Move package
- Define basic tontine structure
- Prepare for contribution & payout logic

## Tech Stack
- Sui blockchain
- Move language
